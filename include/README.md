# Include 目录

此目录用于存放项目头文件（.h文件）。

## 文件说明

- `config.h` - 硬件引脚和全局配置定义
- `DisplayManager.h` - 显示管理器接口
- `FaceRenderer.h` - 表情渲染器接口
- `ClockRenderer.h` - 时钟渲染器接口
- `SysInfoRenderer.h` - 系统信息渲染器接口
- `TouchManager.h` - 触摸管理器接口
- `WiFiMgr.h` - WiFi管理器接口
- `TimeManager.h` - 时间管理器接口
- `ConfigManager.h` - 配置管理器接口
- `SystemMonitor.h` - 系统监控接口
